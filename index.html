zepto.js — the aerogel-weight mobile javascript framework

About

Zepto.js is a minimalist inlinable framework for mobile WebKit browsers, with a jQuery-like chaining syntax.

Our goal: ~2k library that handles most basic dredge work for you in a nice API so you can concentrate on getting stuff done.

Follow our progress on Twitter @zeptojs. Zepto.js is released under the MIT license.
